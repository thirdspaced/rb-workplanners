# AL2/AL3 Work Plan Generator

Generate personalized work plans for AL2 and AL3 students from Excel task trackers.

## Features

- ✅ Extracts tasks based on student-teacher assignments
- ✅ Handles Lexia units with checkboxes
- ✅ Day indicators for AL3 students
- ✅ Generates 38 individual Word documents
- ✅ Batch download for all AL2 or AL3 students

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/work-plan-generator)

## Usage

1. Upload Task Tracker Excel file
2. Upload Content Groups Excel file
3. Enter week number
4. Click "Generate Work Plans"
5. Download individual or batch files
